# Blinkit Sales Dashboard – Power BI Project

## 📌 Project Overview
This project is an interactive **Blinkit Sales Dashboard** built using **Power BI** to analyze sales performance, outlet efficiency, product categories, and customer ratings. The dashboard provides actionable business insights through dynamic visualizations and KPIs, helping stakeholders make data-driven decisions.

## 🎯 Project Objectives
- Analyze overall sales performance.
- Track average sales and customer ratings.
- Compare outlet performance across different locations and sizes.
- Identify top-performing product categories.
- Enable interactive data exploration using filters.

## 🛠️ Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning & Transformation
- Data Visualization

## 📊 Key Performance Indicators (KPIs)

| Metric | Value |
|----------|----------|
| Total Sales | $1.20M |
| Average Sales | $141 |
| Number of Items | 8,523 |
| Average Rating | 3.9 |

## 📈 Dashboard Features

### Sales Analysis
- Total sales overview
- Fat content-wise sales analysis
- Sales trend by outlet establishment year

### Product Analysis
- Item type-wise sales distribution
- Top-performing product categories
- Revenue contribution by category

### Outlet Analysis
- Sales by outlet size
- Sales by outlet location tier
- Outlet type performance comparison

### Interactive Filters
Users can filter the dashboard by:
- Outlet Location Type
- Outlet Size
- Item Type

## 🔍 Key Insights
- Tier 3 outlets generated the highest revenue.
- Medium-sized outlets contributed the largest share of sales.
- Fruits, Snacks, Household, and Frozen Foods are among the best-selling categories.
- Supermarket Type 1 recorded the highest sales performance.
- Low Fat products generated a significant portion of total revenue.

## 🚀 Business Impact
This dashboard helps businesses:
- Monitor sales performance in real-time.
- Identify profitable outlet segments.
- Optimize inventory planning.
- Support strategic decision-making through data visualization.
